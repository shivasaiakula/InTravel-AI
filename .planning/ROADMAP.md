# Roadmap

## Proposed Roadmap

**2 phases** | **10 requirements mapped** | All v1 requirements covered ✓

| # | Phase | Goal | Requirements |
|---|-------|------|--------------|
| 1 | Core & Auth | Finalize authentication, destination browsing, and basic AI trip generation. | AUTH-01, AUTH-02, TRIP-01, TRIP-02, DEST-01 |

### Phase Details

**Phase 1: Core & Auth**
Goal: Finalize authentication, destination browsing, and basic AI trip generation.
Requirements: AUTH-01, AUTH-02, TRIP-01, TRIP-02, DEST-01
Success criteria:
1. User can register and log in successfully.
2. User can request and view an AI-generated itinerary.
3. User can browse destinations.

**Phase 2: Ecosystem & Bonus**
Goal: Complete transport, hotels, and add bonus integrations (Weather, Charts, Maps).
Requirements: TRAN-01, HOTL-01, FEAT-01, FEAT-02, FEAT-03
Success criteria:
1. User can view live weather data on a destination page.
2. User sees budget visualization charts for generated trips.
3. User can query transport and hotels with mock/real data returning properly.
